Projects

InFit

InFit is a social media web app that lets you store, rate, categorize, and share clothes in your wardrobe. This project was built with a group of 3 other incredible engineers using MongoDB, Express.JS, React/Redux, and Node.JS.

In the role of Team Leader, I was responsible for directing the look, feel, and functionality of the product as well as planning and delegating daily tasks for our team. We were able to implement drag and drop technology with the "Beautiful DnD" library for React. InFit's mascot, the InFit Impala, will forever be in my heart.

KloudClone

KloudClone is a clone of the popular music-sharing website SoundCloud. Built with Ruby on Rails and React/Redux.

This project was really fun to work on because I was able to combine my passion for music with my passion for coding. Figuring out how to leverage React and Redux states in combination with the audio HTML element to develop a stylish and fully functional music-player was a fun challenge. The biggest struggle, though, was having to not completely jam-out every time I was testing my music-player.
